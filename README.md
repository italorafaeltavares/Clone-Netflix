 Dependências
- NodeJS >= 14 LTS

Realize o clone deste projeto:
````
git clone https://github.com/italorafaeltavares/Clone-Netflix.git
````
Acesse o projeto:
````
cd Clone-Netflix
````
Utilize seu terminal para intalar o express:
````
npm i express
````
Acesse o Netflix Clone:
````
http://localhost:3333

````
